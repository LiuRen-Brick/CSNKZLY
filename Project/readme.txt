The project is a simple example for FMD Microelectronics 'FT32F072' microcontroller 
using FMD FT32F072-Starter-kit Board, compliant to Cortex Microcontroller Software 
Interface Standard (CMSIS).

Example functionality:
 - Clock Settings:
   - XTAL    =            8.00 MHz
   - SYSCLK  =           48.00 MHz
   - HCLK    = SYSCLK  = 48.00 MHz

--SysTick
